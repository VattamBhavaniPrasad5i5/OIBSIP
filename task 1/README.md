

# Iris Flower Classification 
This project implements the classification of the famous Iris flower dataset using two different algorithms: Logistic Regression and GaussianNB. The goal of this project is to compare the performance of the two algorithms and determine which one provides better accuracy.

# Dataset
The Iris flower dataset is a multivariate dataset introduced by the British statistician and biologist Ronald Fisher in 1936. The dataset consists of 50 samples from each of three species of Iris flowers (Iris setosa, Iris virginica, and Iris versicolor), for a total of 150 samples. For each sample, four features are measured: sepal length, sepal width, petal length, and petal width.

# Requirements

- Python 3.x
- Scikit-learn
- NumPy
- Pandas
- Matplotlib

# Results
After running the script, the accuracy of both algorithms will be displayed. Based on our experiments, Logistic Regression gives higher accuracy on this dataset than GaussianNB.

# Conclusion
In conclusion, we have successfully classified the Iris flower dataset using two different algorithms: Logistic Regression and GaussianNB. Based on our experiments, Logistic Regression provided better accuracy than GaussianNB. This project can be used as a starting point for more complex classification problems involving larger datasets.
